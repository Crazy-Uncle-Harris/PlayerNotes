This plugin allows you and your admins to keep tabs on people easily.
All data is stored in SQL for fast access.

Command list

/pnotesview <PLAYERNAME>
Show notes about a player. Alias: /pnv

/pnotesadd <PLAYERNAME> <NOTE>
Add a note. Alias: /pna

/pnotesposted <PLAYERNAME>
Show notes from a person. Alias /pnp
