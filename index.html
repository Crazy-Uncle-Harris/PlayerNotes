PlayerNotes

This plugins allows you and your admins to keep tabs on people easely.
All the data are stored in SQL (only) for really fast access with no problems. We hope soon enough to have ready a WebUI for it.

Command list

/pnotesview <PLAYERNAME>
Show notes about a player. Alias: /pnv
/pnotesadd <PLAYERNAME> <NOTE>
Add a note. Alias: /pna
/pnotesposted <PLAYERNAME>
Show notes from a person. Alias /pnp
